CSS BASICS

Introduction to CSS

CSS (Cascading Styles Sheet) is the code that styles a webpage. For example decorating the background with a certain color or image.

So let's talk a bit more about CSS

Like HTML, CSS and neither is it a programming language. It is a style sheet.
It is basically used to style HTML elements.

Anatomy of a CSS Ruleset

The above structure is called a Ruleset. Though it is mostly referred to as just rule

the following are the main parts of a CSS ruleset:

  1. Selector: This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled (in this example, "p" elements)
  2. Declaration:This is a single rule like color: red;. It specifies which of the element's properties you want to style.
  3. Properties:These are ways in which you can style an HTML element. (In this example, color is a property of the "p" elements.) In CSS, you choose which properties you want to affect in the rule.
  4. Property value:To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many color values in addition to red.)
